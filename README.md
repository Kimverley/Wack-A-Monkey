# Whack-a-Monkey

Whack-a-Monkey is a fun arcade-style browser game where monkeys pop out of barrels and players must quickly hit them to earn points while avoiding bomb monkeys.

## Features

- Fast-paced gameplay
- Random monkey appearances
- Bomb monkeys to avoid
- Score tracking system
- Increasing difficulty over time
- Simple and responsive controls

## How to Play

1. Start the game in your browser
2. Watch for monkeys popping out of barrels
3. Click or tap monkeys to score points
4. Avoid monkeys carrying bombs
5. Try to achieve the highest score possible

## Technologies Used

- HTML
- CSS
- JavaScript
